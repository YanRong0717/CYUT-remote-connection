## 租用介面
![Borrow UI.](/figure/borrow_UI.png)
* 點選左上角 ( 時鐘icon ) 圖案可查看當前開放時間
* 點選姓名右邊 ( 鉛筆icon ) 可編輯使用者密碼
* 點選右上角 ( 箭頭icon ) 登出當前使用者
* 點選下方 Borrow 按鈕即可租借電腦

## 後台管理介面
